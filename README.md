# ☕ Coffee Shop Website

A modern, responsive, and visually appealing Coffee Shop Website built using **HTML, CSS, and JavaScript**. The website provides an engaging user experience with an attractive landing page, smooth navigation, menu showcase, customer testimonials, image gallery, and contact section.
